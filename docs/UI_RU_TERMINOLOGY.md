# Image Studio RU terminology

Дата: 2026-06-19

Цель: держать русский интерфейс единообразным и не смешивать пользовательские понятия с техническими API-терминами.

## Пользовательские термины

| English/source | RU UI term | Примечание |
|---|---|---|
| composer | панель запроса | Нижняя панель, где пользователь пишет описание и добавляет вложения. |
| controls | меню действий / действия | Не использовать как самостоятельное английское слово в пользовательском тексте. |
| image wall | галерея | Экран результатов. |
| batch | очередь / пакетная генерация | `Очередь` для коротких labels, `пакетная генерация` для описаний. |
| prompt | описание / промпт | В техническом контексте допустим `промпт`; в onboarding лучше `описание`. |

## Технические термины

| Term | Как писать |
|---|---|
| endpoint | endpoint / endpoint-адрес |
| payload | технический JSON / payload в API-контексте |
| model ID | model ID / технический ID модели |
| raw JSON | raw JSON |
| API key | API-ключ |

Технические термины можно оставлять английскими там, где это имя поля, протокольное понятие или часть API-настройки.
