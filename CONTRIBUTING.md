# Contributing

Contributions are welcome.

## Local setup

```bash
npm install
npm run dev
```

Before opening a pull request, run:

```bash
npm run build
```

## Notes

- Keep the app local-first and safe by default.
- Do not commit generated output, local databases, real provider keys, or personal `.env` files.
- Prefer small, focused pull requests with a short explanation of the UI or API behavior being changed.
